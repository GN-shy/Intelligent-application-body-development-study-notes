#  响应网页开发

响应式布局是一种网页设计方法，使网页能够根据用户设备的屏幕尺寸，分辨率，方向（横屏/竖屏）等特性自动调整布局和内容，提供一致且优化的用户体验，其核心目标是“一次设计，多端适配”



| 优点                    | 缺点                                    |
| ----------------------- | --------------------------------------- |
| 1. 一套代码适配多端设备 | 1. 复杂布局需较多调试                   |
| 2. 节省开发和维护成本   | 2. 过度使用媒体查询可能影响性能         |
| 3. 用户体验一致性高     | 3. 需精准计算断点(什么屏幕下，怎么显示) |

核心技术：

布局方式：

- 弹性盒子（Flexbox）
- 网格布局（CSS Grid）
- 百分比布局(流式布局)
- 单位可以借助于vw实现

断点（Breakpoints）设置：

- 根据常见设备宽度设置断点

- 手机 < 768px

- 平板 768px~1024px

- PC > 1024px

- 根据不同设备宽度显示不同布局样式

  ​

  | 断点名称 | 最小宽度  | CSS 媒体查询                 | 适用场景              |
  | -------- | --------- | ---------------------------- | --------------------- |
  | xs       | &lt;576px | `@media (min-width: 0)`      | 超小设备（手机竖屏）  |
  | sm       | ≥576px    | `@media (min-width: 576px)`  | 平板竖屏/小屏手机横屏 |
  | md       | ≥768px    | `@media (min-width: 768px)`  | 平板横屏/桌面小屏     |
  | lg       | ≥992px    | `@media (min-width: 992px)`  | 桌面中屏              |
  | xl       | ≥1200px   | `@media (min-width: 1200px)` | 桌面大屏              |
  | xxl      | ≥1400px   | `@media (min-width: 1400px)` | 超宽屏（4K 显示器等） |

## 媒体查询

媒体查询是CSS3的功能，允许根据设备的特性（如屏幕宽度，高度，分辨率，设备类型等）应用不同的样式规则，是实现响应式设计的核心技术。

语法

```markdown

@media 媒体类型 and (媒体特征) {
    /* 样式 */
}
```

- `@media`：媒体查询的起始声明
- 设备类型：媒体类型（可选），如 `screen`, `print` 等，省略默认 `all`
- 设备特性：如 `min-width`, `max-width` 等

# 4.1 断点：

断点（Breakpoints）是响应式设计的组成部分。通过断点可以控制何时在特定视口（viewport）或设备尺寸下调整布局。

表格

| Breakpoint        | Class infix | Dimensions |
| ----------------- | ----------- | ---------- |
| Extra small       | None        | <576px     |
| Small             | `sm`        | ≥576px     |
| Medium            | `md`        | ≥768px     |
| Large             | `lg`        | ≥992px     |
| Extra large       | `xl`        | ≥1200px    |
| Extra extra large | `xxl`       | ≥1400px    |



Bootstrap附带三种不同的容器：
- `.container`，它在每个响应断点处设置 `max-width`
- `.container-fluid`，所有断点 `width: 100%`
- `.container-{breakpoint}`，直到指定的断点 `width: 100%` 为止

## 4. 布局（Layout）

4.3 网格系统（grid）

Bootstrap 的网格系统（栅格系统）使用一系列容器、行和列来布局和对齐内容。它是用 flexbox 构建的，并且是完全响应的。

> **注意：** Bootstrap 的网格系统每行划分为 **12 份**。

网格示意

| 示例1（等分布局）                        | 示例2（非等分布局）           |
| :--------------------------------------- | :---------------------------- |
| `col` + `col` + `col` + `col`（4列均分） | `col-8` + `col-4`（8:4 分配） |

响应式断点与类名前缀

| 设备尺寸       | 断点       | `container` 最大宽度 | 类名前缀    | 列数 |
| :------------- | :--------- | :------------------- | :---------- | :--- |
| 超小屏幕 (xs)  | `< 576px`  | None (auto)          | `.col-`     | 12   |
| 小屏幕 (sm)    | `≥ 576px`  | 540px                | `.col-sm-`  | 12   |
| 中等屏幕 (md)  | `≥ 768px`  | 720px                | `.col-md-`  | 12   |
| 大屏幕 (lg)    | `≥ 992px`  | 960px                | `.col-lg-`  | 12   |
| 超大屏幕 (xl)  | `≥ 1200px` | 1140px               | `.col-xl-`  | 12   |
| 特大屏幕 (xxl) | `≥ 1400px` | 1320px               | `.col-xxl-` | 12   |

代码示例

```html
<div class="container">
  <div class="row">
    <div class="col-md-3">1</div>
    <div class="col-md-3">2</div>
    <div class="col-md-3">3</div>
    <div class="col-md-3">4</div>
  </div>
</div>
```

注意事项

1.  `row` 代表一行
2.  `col-xx-份数` 格式，例如 `col-md-3`
3.  可以写多个类名，决定不同宽度下的排列个数
