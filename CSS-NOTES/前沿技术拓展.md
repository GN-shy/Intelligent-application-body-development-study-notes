## SVG动画

SVG是一种基于XML的矢量图形标准，由W3C制定，支持无损缩放，交互性和动态效果。

其核心特点包括：

1. **矢量特性**：无论放大或缩小均保持清晰；文件体积小，适合网络传输，尤其适用于高分辨率设备。
2.  **可编辑性**：直接通过文本编辑器修改 SVG 代码，支持颜色、形状、动画参数的实时调整。
3.  **交互性**：支持 JavaScript 和 CSS 控制，可实现点击、悬停等动态响应。
4.  **兼容性**：主流浏览器（Chrome、Firefox、Safari）均原生支持 SVG，移动端适配性强。

> svg组成
>
> - SVG根元素。SVG图标必须包裹在< svg >标签内
> - < path >路径，通过`d`属性定义路径指令。 M：移动画笔到坐标点   Z： 闭合路径
>
> 在iconfont中下载，下载素材，svg格式，打开后查看源代码，svg部分复制

> svg图标常见CSS属性：
>
> svg是行内块元素类似，可以设置大小，移动位置，动画等。但是有自己特殊的样式属性。
>
> | 属性                | 描述                                                | 取值示例                  |
> | ------------------- | --------------------------------------------------- | ------------------------- |
> | `fill`              | 填充颜色（支持颜色值、渐变、图案）不需要改为 `none` | `fill: #f00;`             |
> | `stroke`            | 定义描边颜色（支持颜色值、渐变、图案）              | `stroke: #f00;`           |
> | `stroke-width`      | 描边宽度（支持像素、百分比、`px`、`em` 等单位）     | `stroke-width: 2px;`      |
> | `stroke-dasharray`  | 虚线模式（实线长度 + 间隔长度）                     | `stroke-dasharray: 10;`   |
> | `stroke-dashoffset` | 调整虚线与间隔的起始位置                            | `stroke-dashoffset: 100;` |
>
> - stroke-dasharray: 100; 默认px  实线和间隔都是 100
>
> - stroke-dasharray: 100 50;  第一个值是实线, 第二个是间隔
> - stroke-dashoffset 用于调整虚线模式的起始偏移量，增大stroke-dashoffset,让虚线模式向路径起点移动，视觉上像线条被擦除







## clip-path 裁剪

clip-path创建复杂的裁剪形状，使元素仅显示被裁剪区域内的部分

clip-path: 内置几何形。

- 内置几何形。状如 circle()、polygon()

```css
.circle {
  clip-path: circle(40% at 50% 50%);
}
```

## 背景滤镜

通过`filter`属性实现，可对元素及其子元素进行实时图像处理，无需修改原始素材



| 滤镜函数        | 语法示例                                                     |
| --------------- | ------------------------------------------------------------ |
| `blur()`        | 高斯模糊 `blur(5px)`                                         |
| `brightness()`  | 调整元素亮度 `brightness(150%)`<br>100%（或 1）为原始亮度；<br>- &lt;100%（或 &lt;1）变暗；<br>- &gt;100%（或 &gt;1）变亮。 |
| `contrast()`    | 调整元素对比度 `contrast(200%)`<br>同上                      |
| `saturate()`    | 调整背景饱和度<br>取值 0%（无色彩）~ 100%（原饱和度）~ &gt;100%（更高饱和）。 |
| `grayscale()`   | 将元素转换为灰度图 `grayscale(100%)`<br>- 0%（或 0）为原始色彩；- 100%（或 1）为完全灰度。 |
| `hue-rotate()`  | 调整元素色相（改变颜色倾向）`hue-rotate(90deg)`              |
| `sepia()`       | 将元素转换为深褐色（复古效果）`sepia(70%)`<br>- 0%（或 0）为原始色彩；<br>- 100%（或 1）为完全深褐色。 |
| `drop-shadow()` | 为元素添加投影 类似 box-shadow，但支持非矩形元素。`drop-shadow(5px 5px 5px #669)` |

**backdrop-filter** 是 CSS 中的一个图形滤镜属性，主要用于对元素背后的区域（即“背景”）应用滤镜效果。

它通常与半透明背景配合使用，能实现类似“毛玻璃”的视觉效果。

# 核心区别

 filter：给【自己本身】加滤镜

- 影响**当前元素**（图片、文字、背景、边框… 全部变色 / 变模糊）
- 自己 + 内容一起变

**backdrop-filter：给【背后的东西】加滤镜**

- 影响**元素后面的内容**
- **自己必须是半透明 / 透明**才能看见效果
- 最经典：**毛玻璃效果（磨砂玻璃）**


## 动画时间线

animation-timeline用于控制动画时间线的核心属性，允许开发者将动画进度和特定事件（如滚动，视口可见性）绑定，从而实现更复杂的交互效果

> 滚动时间线：动画进度与页面或容器的滚动位置关联
>
> ```
> /* 滚动进度条 - 使用 CSS 滚动时间线实现 */
> .scrollbar {
>   width: 0%;
>   height: 3px;
>   background: linear-gradient(90deg, /* 这里可以补充颜色值，例如 #4299e1, #38b2ac */);
>   animation: move 2s;
>   /* 绑定时间线 */
>   animation-timeline: scroll();
> }
>
> @keyframes move {
>   0% {
>     width: 0;
>   }
>   100% {
>     width: 100%;
>   }
> }
> ```

> 视图时间线：动画进度与元素进入/离开视口的可见性关联
>
> ```
> .box img {
>   opacity: 0.2;
>   animation: scale 1s linear forwards;
>   animation-timeline: view();//绑定
> }
>
> @keyframes scale {
>   0% {
>     transform: scale(1);
>     opacity: 0.2;
>   }
>
>   100% {
>     transform: scale(1.5);
>     opacity: 1;
>   }
> }
> ```
>

## 变量和计算函数

 CSS增强了逻辑性和动态计算能力，比如CSS变量和一些动态函数（calc(),clamp()等）

使用场景：

- 主题切换
- 响应式设计
- 交互动画等

> 变量
>
> CSS变量（也称为自定义属性）是一种动态存储和复用值的机制。让样式管理更灵活，可维护。
>
> 变量可以理解为一个容器，里面可以存放数据
>
> 
>
> 定义变量：
>
> 通过--变量名 定义变量
>
> ```
> --color: 
> ```
>
> 使用变量
>
> 通过var(--变量名）使用定义的变量
>
> ```
> color: var(--color);
> background-color: var(--bgcolor);
> ```
>
> 
>
> 作用域：变量的作用范围，哪个区域有效
>
> **定义变量**
>
> ```markdown
>
> :root { /* 全局变量 */
>     --color: #000;
> }
>
> .box { /* 局部变量 */
>     --bgcolor: pink;
> }
> ```
>
> 变量引用示例（使用 `var()`）
>
> ```css
> :root {
>     --brand_pink: var(--Pi5);
>     --brand_pink_thin: var(--Pi1);
>     --brand_blue: var(--Lb5);
>     --brand_blue_thin: var(--Lb1);
>     --stress_red: var(--Re5);
>     --stress_red_thin: var(--Re1);
> }
> ```
>
> - 全局作用域：通过 `:root` 定义，作用于整个网页。
>
> - 局部作用域：在特定选择器中定义，仅影响该元素及其子元素。
> **使用变量**
>
> ```
>
> .nav {
>     color: var(--color);
>     background-color: var(--bgcolor);
> }
> ```
>
> 通过var（--变量名）使用定义的变量
>
> ```
> #i_cecream {
>     margin: 0 auto;
>     max-width: 2560px;
>     background-color: var(--bg1);
> }
> ```



> calc()函数
>
> calc（）函数可以执行数学运算，支持混合单位
>
> 语法：
>
> ```
> .box{
>     width: calc(100%-20px); //父宽度减固定值
>     height: calc(var(--base-size)*1.5); //变量与运算结合
> }
> ```
>
> 加减乘除运算写法：+- * /注意符号左右两侧保留空格
>
> 使用场景：
>
> - 不同单位混合计算（% + px、vw - px）
> - 变量 + - * / 计算
> - 自适应留间距
